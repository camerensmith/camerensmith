Projects

Fun projects I work on in my free time. Helps me learn new things and stay current on webtech.

  • Gwent Electron

    Playable desktop (x64) expansion of a browser remake of the Witcher 3 Gwent minigame, fork of 3.1 with more assets and better AI.

  • OWTacticards

    Complete overhaul of the original digitisation. Now with AI decision making, new cards, sounds, modal choices, unique abilities, and more. Credit to Nathan Miles and barberian for initial iterations.

  • ProjectLookbook (Mobile Alpha Demo)

    Mobile-first closet organizer with background removal, outfit builder, and Firebase sync. Full authentication and offline support.

  • Search Builder

    Helps myself and other jobseekers search for their dream job smarter, all in one place, with only a few clicks. No frills.

  • OpenEco Project

    100% volunteer-run, central database to emphasize truth in reporting for corporate climate initiatives.

  • Quartz

    Cross-platform desktop application for managing personal databases. Features collections management, spreadsheet-like table view, customizable forms, full-text search, and a plugin system.

  • SuperMauLTD (Alpha)

    Experimental TD project based on old WC3 maps. Built from the ground up in Python with highly modular json files. This game is unfinished. You will encounter bugs, unintuitive UI, non functional issues.
