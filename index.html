Salesforce App Case Study

Objective: Integrate a custom Salesforce app to send to dev team. Learn how to use and integrate UML concepts and SQL in a realistic setting. Proof of concept project.

Project Overview

This proof of concept project involved creating a comprehensive design specification for a custom Salesforce application. I gathered user requirements and translated them into a detailed design spec for the development team, capturing the initial business issue and desired outcomes.

Key Deliverables:
  • Requirements Analysis: Defined use-cases, acceptance criteria, and non-functional requirements
  • Technical Documentation: Used class diagrams and UML for SQL mapping, outlined backend database process flow
  • SDLC Planning: Documented general SDLC roadmap and overview using Visio
